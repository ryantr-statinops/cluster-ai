// 02 — Architecture

Mô hình kiến trúc 3 lớp

Thiết kế tách biệt rõ ràng giữa Core Engine, Business Logic và Interface Layer — đảm bảo khả năng mở rộng và bảo trì dài hạn.

Layer 01

CORE

Nền tảng hạ tầng: Database, Message Queue, AI Model Runtime, Authentication. Chạy độc lập, không phụ thuộc UI.

PostgreSQL Redis Model Runner Auth Service
Layer 02

LOGIC

AI Orchestrator, Workflow Engine, Business Rules. Nơi các Agent được điều phối và quy trình được thực thi theo logic nghiệp vụ.

Agent Manager Rule Engine API Gateway
Layer 03

INTERFACE

Web Dashboard, Mobile App, External Integrations (Google Sheets, Slack, Email). Lớp người dùng tương tác trực tiếp.

Web App REST API Webhooks
